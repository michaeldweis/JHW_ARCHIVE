John H. Weis Archive

A preserved collection of line-printer graphics, handwritten programming notes, utility listings, provenance records, and early computational folk-art materials connected to John H. Weis.

View Gallery Download Full Report
Archive Significance

The collection preserves process and output together: code references, card/tape workflows, EBCDIC/Hollerith materials, printer instructions, and finished character-based images.

It can be read as both systems-programming evidence and early machine-room visual culture.

What the Archive Contains

Printer Art

Character-based images including Edith, Snoopy/Peanuts figures, Alfred E. Neuman, Madonna and Child, reindeer, bells, and other demonstration pieces.

Programming Evidence

Handwritten flowcharts, tape-to-printer instructions, card-number changes, branching notes, and System/360-era utility materials.

Provenance

Documents and certificates help place John H. Weis within a broader technical training and computing context.

Technical Reading

The images appear to have been generated by batch-oriented printer routines that converted picture data into rows of fixed-width characters. Character choice, density, repetition, and overstrike-like effects created tonal variation.

Art-Historical Reading

The archive fits a broader category of early computational visual culture: work made within technical workplaces using available machines, shared imagery, seasonal humor, and procedural image construction.
